## **Stats**

This contains stats which define features of a character.

### _Title_

- Name
- Level

### _Base Stats_

- Strength
- Stamina
- Intelligence
- Agility
- Luck

### _Derived Stats_

- Health
- Mana
- Attack
- Defence
- Mana Attack
- Mana Defence
- Speed
- Accuracy
- Evasion
- Crit Chance

### _Other Data_

- Current Exp.
- Exp. to Next Level

### _Enemy Data_

- Exp. Drop
- Money Drop
